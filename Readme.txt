Algorithme de recommendation :

 - Un système de tags est proposé. Une fois le tag coché, seules apparaissent les offres contenant ce tag
dans leur titre ou leur body.
En cochant ces tags non sur la page d'accueil mais sur son compte, l'utilisateur enregistre ses choix pour ses 
prochaines visites.
Sur son compte, il peut également ajouter des tags personnalisés.

- L'utilisateur peut ajouter une offre en favori, ainsi, il pourra la retrouver dans la page dédiée à ses offres 
favorites.

- En plus des tags, deux modes de tri sont proposés.
	L'utilisateur peut classer les offres par pertinence, ainsi les offres ayant été le plus mises en favori sur
l'emsemble des utilisateurs apparaitront en haut de liste (l'utilisateur du critère de recherche par les favoris pour 
éviter la mise en concurrence).
	L'utilisateur peut classer les offres par date, dans l'ordre croissant ou décroissant.
